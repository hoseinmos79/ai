# افزونه آزمون آنلاین با پرداخت (وردپرس)

این افزونه امکان برگزاری آزمون‌های چهارگزینه‌ای آنلاین را با قابلیت پرداخت از طریق زرین‌پال و کارت به کارت فراهم می‌کند.

- ✅ مشاهده نتایج و عملکرد کاربران

## نصب و راه‌اندازی

### مرحله 1: نصب افزونه
1. فایل‌های افزونه را در پوشه `/wp-content/plugins/online-exam-payment/` قرار دهید
2. از پنل مدیریت وردپرس، افزونه را فعال کنید
3. پس از فعال‌سازی، جداول پایگاه داده به صورت خودکار ایجاد می‌شوند

### مرحله 2: تنظیمات اولیه
1. به منوی **آزمون‌های آنلاین > تنظیمات** بروید
2. اطلاعات زرین‌پال را وارد کنید:
   - مرچنت آی‌دی
   - فعال/غیرفعال کردن حالت آزمایشی
3. اطلاعات کارت به کارت را تنظیم کنید:
   - شماره کارت
   - نام صاحب حساب
   - توضیحات واریز
4. واحد پول (تومان/ریال) را انتخاب کنید
5. پیام‌های موفقیت و خطا را سفارشی کنید

### مرحله 3: ایجاد آزمون
1. به **آزمون‌های آنلاین > افزودن آزمون جدید** بروید
2. عنوان و توضیحات آزمون را وارد کنید
3. تنظیمات آزمون را مشخص کنید:
   - قیمت آزمون (تومان)
   - مدت زمان (دقیقه)
   - حدنصاب قبولی (درصد)
   - حداکثر تعداد تلاش
   - نمایش نتایج
   - ترتیب تصادفی سوالات
4. آزمون را منتشر کنید

### مرحله 4: افزودن سوالات

#### روش 1: وارد کردن از فایل
1. به **آزمون‌های آنلاین > وارد کردن سوالات** بروید
2. آزمون مورد نظر را انتخاب کنید
3. فایل سوالات را آپلود کنید (فرمت‌های پشتیبانی شده: .txt, .doc, .docx)

**فرمت فایل سوالات:**
```
سوال نمونه: کدام گزینه صحیح است؟
الف) گزینه اول
ب) گزینه دوم
*ج) گزینه سوم (پاسخ صحیح)
د) گزینه چهارم

سوال دوم: این سوال دوم است؟
ا) پاسخ اول
*ب) پاسخ صحیح
ج) پاسخ سوم
د) پاسخ چهارم
```

#### روش 2: افزودن دستی
1. به **آزمون‌های آنلاین > افزودن سوال جدید** بروید
2. عنوان و متن سوال را وارد کنید
3. چهار گزینه را تکمیل کنید
4. پاسخ صحیح را انتخاب کنید
5. در صورت نیاز، توضیح اضافی وارد کنید
6. آزمون مربوطه را انتخاب کنید

### مرحله 5: استفاده از شورت‌کدها

#### نمایش لیست آزمون‌ها
```

```

#### نمایش یک آزمون خاص
```

```

#### پنل کاربری
```

```

#### فرم پرداخت
```

```

## مدیریت تراکنش‌ها

### پرداخت‌های آنلاین (زرین‌پال)
- پس از پرداخت موفق، دسترسی به آزمون به صورت خودکار فعال می‌شود
- تراکنش‌ها با کد پیگیری زرین‌پال ثبت می‌شوند

### پرداخت‌های کارت به کارت
1. کاربر اطلاعات واریز را ثبت می‌کند
2. در پنل **تراکنش‌ها**، تراکنش در وضعیت "در انتظار" قرار می‌گیرد
3. مدیر پس از بررسی، تراکنش را تایید یا رد می‌کند
4. پس از تایید، دسترسی کاربر به آزمون فعال می‌شود

## مشاهده نتایج

### برای مدیران
- **آزمون‌های آنلاین > نتایج آزمون‌ها**
- فیلتر بر اساس آزمون
- مشاهده جزئیات پاسخ‌های هر کاربر
- امکان صدور گزارش

### برای کاربران
- پنل کاربری > نتایج آزمون‌ها
- مشاهده نمره و وضعیت قبولی
- بررسی پاسخ‌های صحیح و اشتباه
- تاریخچه تمام آزمون‌ها

## پیکربندی پیشرفته

### تنظیمات پایگاه داده
افزونه جداول زیر را ایجاد می‌کند:
- `wp_oep_exam_results`: نتایج آزمون‌ها
- `wp_oep_transactions`: تراکنش‌های پرداخت
- `wp_oep_exam_sessions`: جلسات آزمون

### هوک‌ها و فیلترها
```php
// تغییر پیام موفقیت پرداخت
add_filter('oep_payment_success_message', function($message, $exam_id) {
    return 'پیام سفارشی شما';
}, 10, 2);

// اعمال تخفیف
add_filter('oep_exam_price', function($price, $exam_id, $user_id) {
    // منطق تخفیف
    return $price * 0.8; // 20% تخفیف
}, 10, 3);
```
