# افزونه آزمون آنلاین با پرداخت

افزونه جامع وردپرس برای برگزاری آزمون‌های چهارگزینه‌ای آنلاین با امکان پرداخت از طریق زرین‌پال و کارت به کارت

## ویژگی‌ها

### پنل مدیریت
- **مدیریت آزمون‌ها**: ایجاد و ویرایش آزمون‌ها با تنظیمات کامل (قیمت، مدت زمان، نمره قبولی، تعداد تلاش)
- **مدیریت سوالات**: افزودن سوالات به صورت دستی یا وارد کردن از فایل Word
- **وارد کردن خودکار**: پشتیبانی از فایل‌های `.docx`، `.doc` و `.txt`
- **دسته‌بندی**: مدیریت دسته‌بندی‌های آزمون‌ها
- **تنظیمات پرداخت**: پیکربندی زرین‌پال و پرداخت کارت به کارت
- **مدیریت تراکنش‌ها**: بررسی و تایید پرداخت‌های دستی
- **گزارش نتایج**: مشاهده نتایج تمامی کاربران
- **تنظیمات عمومی**: سفارشی‌سازی پیام‌ها و تنظیمات کلی

### پنل کاربری
- **فهرست آزمون‌ها**: مشاهده آزمون‌های موجود با امکان فیلتر و جستجو
- **خرید آزمون**: پرداخت آنلاین (زرین‌پال) یا آفلاین (کارت به کارت)
- **شرکت در آزمون**: رابط کاربری مدرن با تایمر و نوار پیشرفت
- **مشاهده نتایج**: نمایش نتایج تفصیلی با پاسخ‌های صحیح
- **تاریخچه**: مشاهده سوابق آزمون‌ها و نتایج قبلی
- **پروفایل**: مدیریت اطلاعات حساب کاربری

### امکانات فنی
- **پشتیبانی کامل RTL**: طراحی مخصوص زبان فارسی
- **سازگاری با المنتور**: ارائه شورت‌کدهای مختلف
- **امنیت بالا**: استفاده از nonce و بررسی دسترسی‌ها
- **عملکرد بهینه**: استفاده از AJAX و کش
- **طراحی واکنش‌گرا**: سازگار با موبایل و تبلت

## نصب و راه‌اندازی

### پیش‌نیازها
- WordPress 5.0 یا بالاتر
- PHP 7.4 یا بالاتر
- MySQL 5.6 یا بالاتر

### مراحل نصب

1. **آپلود افزونه**:
   ```bash
   # آپلود فایل‌های افزونه به پوشه wp-content/plugins/online-exam-payment/
   ```

2. **فعال‌سازی**:
   - به بخش افزونه‌ها در پنل مدیریت بروید
   - افزونه "آزمون آنلاین با پرداخت" را فعال کنید

3. **تنظیمات اولیه**:
   - به منوی "آزمون آنلاین" > "تنظیمات" بروید
   - اطلاعات زرین‌پال خود را وارد کنید
   - اطلاعات کارت بانکی را برای پرداخت دستی تنظیم کنید

## راهنمای استفاده

### ایجاد آزمون جدید

1. به "آزمون آنلاین" > "آزمون‌ها" بروید
2. روی "افزودن آزمون جدید" کلیک کنید
3. اطلاعات آزمون را وارد کنید:
   - عنوان و توضیحات
   - قیمت (0 برای رایگان)
   - مدت زمان (0 برای بدون محدودیت)
   - نمره قبولی (درصد)
   - حداکثر تعداد تلاش

### افزودن سوالات

#### روش دستی:
1. به "آزمون آنلاین" > "سوالات" بروید
2. "افزودن سوال جدید" را کلیک کنید
3. متن سوال و گزینه‌ها را وارد کنید
4. پاسخ صحیح را انتخاب کنید
5. آزمون مربوطه را انتخاب کنید

#### وارد کردن از فایل:
1. به "آزمون آنلاین" > "وارد کردن سوالات" بروید
2. آزمون مقصد را انتخاب کنید
3. فایل Word یا متنی خود را آپلود کنید
4. روی "وارد کردن" کلیک کنید

**فرمت فایل نمونه:**
```
سوال اول: کدام گزینه صحیح است؟
الف) گزینه اول
ب) گزینه دوم
*ج) گزینه سوم (صحیح)
د) گزینه چهارم

سوال دوم: این سوال چگونه است؟
a) First option
*b) Correct option
c) Third option
d) Fourth option
پاسخ: b

توضیح: این توضیح اختیاری است.
```

### تنظیم پرداخت

#### زرین‌پال:
1. به "تنظیمات" بروید
2. تب "پرداخت" را انتخاب کنید
3. مرچنت آی‌دی زرین‌پال را وارد کنید
4. واحد پول را انتخاب کنید (ریال/تومان)
5. حالت sandbox را در صورت تست فعال کنید

#### کارت به کارت:
1. شماره کارت یا شبا را وارد کنید
2. نام صاحب حساب را مشخص کنید
3. توضیحات مربوط به نحوه واریز را بنویسید

### استفاده از شورت‌کدها

#### لیست آزمون‌ها:
```php
[oep_exam_list columns="3" orderby="date" order="DESC"]
```

#### آزمون مشخص:
```php
[oep_exam_single id="123"]
```

#### پنل کاربری:
```php
[oep_user_panel]
```

#### فرم پرداخت:
```php
[oep_payment_form exam_id="123"]
```

### سفارشی‌سازی با CSS

```css
/* تغییر رنگ دکمه‌ها */
.oep-btn-primary {
    background-color: #your-color;
    border-color: #your-color;
}

/* تغییر فونت */
.oep-exam-list, .oep-user-panel {
    font-family: 'YourFont', Tahoma, Arial;
}

/* تغییر رنگ پیام‌ها */
.oep-message-success {
    background-color: #your-success-color;
}
```

## مدیریت پایگاه داده

### جداول ایجاد شده:
- `wp_oep_exam_results`: نتایج آزمون‌ها
- `wp_oep_transactions`: تراکنش‌های پرداخت
- `wp_oep_exam_sessions`: جلسات فعال آزمون

### نوع پست‌های سفارشی:
- `oep_exam`: آزمون‌ها
- `oep_question`: سوالات

### طبقه‌بندی سفارشی:
- `oep_exam_category`: دسته‌بندی آزمون‌ها

## عیب‌یابی

### مشکلات رایج:

1. **خطای 500 هنگام آپلود فایل**:
   - حجم فایل را کاهش دهید
   - مجوزهای پوشه uploads را بررسی کنید

2. **عدم نمایش آزمون‌ها**:
   - تنظیمات permalink را بازنشانی کنید
   - از وجود سوال برای آزمون اطمینان حاصل کنید

3. **مشکل در پرداخت**:
   - اطلاعات زرین‌پال را بررسی کنید
   - SSL سایت را فعال کنید

### فعال‌سازی حالت دیباگ:

```php
// در فایل wp-config.php اضافه کنید
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
```
